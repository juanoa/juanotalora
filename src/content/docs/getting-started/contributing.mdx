¿Has encontrado alguna errata? ¿Crees que hay algo que se puede expresar de forma distinta a como lo he hecho? Estoy abiertos a cambios y a nuevas ideas. Puedes hacerlo de varias formas:

- **Creando un issue**. Si no tienes tiempo o no sabes cómo arreglarlo, puedes abrir un _issue_ explicando el problema.
- **Haciendo un pull request**. Si tienes tiempo y sabes cómo arreglarlo, puedes hacer un _pull request_ con los cambios.
- **Contactando conmigo**. Si no te convence ninguna de las dos opciones anteriores, puedes contactar conmigo a través de [X](https://x.com/juanoa_) o [LinkedIn](https://www.linkedin.com/in/juanoa/).
